# Bauunternehmen Landing Page

Modern landing page for a German construction company built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Features

- Responsive one-page landing page
- Sticky navigation
- Contact form
- SEO-ready structure
- Docker support
- CI/CD ready

## Project Structure

```text
src/
  app/
  components/
  data/
  lib/
  styles/
docs/
public/
```
